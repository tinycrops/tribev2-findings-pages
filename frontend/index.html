Reading a brain model with controlled stimuli

In-silico neuroscience on TRIBE v2 (facebook/tribev2) — Meta's tri-modal foundation model that predicts human fMRI responses · GB10 / Blackwell · 2026-06-15

TRIBE v2 maps video, audio, and text onto the cortical surface (20484 fsaverage5 vertices). We treat it as a subject in an fMRI experiment: feed it controlled stimuli, read out its predicted brain maps, and analyse them like a localizer study — no real brain data in the loop. Three experiments below.

① Language localizer confirmed

Zero-shot, heard speech drives auditory / superior-temporal cortex — the model recovers the textbook localization (Heschl's gyrus, planum temporale) with no training on this task.

Method in one line how

Drive one frozen encoder branch in isolation (video-only / word-only events), read the predicted fsaverage5 map, and compare maps in a common brain space. Every effect is guarded by a null-swap control.

① Heard speech → auditory cortex, zero-shot

Replicating the method of the TRIBE paper's Fig. 5: controlled linguistic stimuli in, predicted fsaverage5 responses out, analysed as a localizer.

② Does a cat-art finetune leave a "cat fingerprint"? no

An ASCII-cat LoRA (reproduced in bf16 from pookie3000/ascii-cats) swapped into the frozen text branch. Minimal-pair stimuli swap only the creature word.

  • Not cat-specific: a roughly uniform ~8% drift across all categories — dog moves more than cat.
  • Lands in language cortex: 62.9% of most-changed vertices are language network (5.4× enrichment), 0.0% visual. Both intuitive hypotheses — cat-selectivity, and a push toward visual cortex because ASCII art is "drawing" — were refuted.
  • Control passed: stock Llama through the exact finetune code path reproduces base to Δ = 0.004% (cosine 1.00000) — the 8% is genuinely the LoRA, not a cache/dtype artifact.

③ Cross-modal: a hint that didn't survive scale

The video branch (V-JEPA 2) reads a real cat photo straight into visual cortex (5.48× enrichment), and a single iconic ASCII cat as text seemed to jump from 0.85× → 2.05× visual under the finetune — an exciting cross-modal bridge.

testfinding
img2ascii realism ladder (bitmap → V-JEPA)holds higher-res ASCII ≈ more cat-like (cos 0.55 full > reduced)
finetune convergence (ASCII-text → image anchor)refuted gain −0.006 across 6 real cats (3/6 positive — null)

The n=1 hint was an artifact of feeding the LoRA exactly the iconic style it was trained on. The honest signal only appeared after scaling with out-of-distribution inputs — a single in-distribution probe can badly over-promise.  Full report →
