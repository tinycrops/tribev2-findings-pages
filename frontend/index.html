Reading a brain model with controlled stimuli

In-silico neuroscience on TRIBE v2 (facebook/tribev2) — Meta's tri-modal foundation model that predicts human fMRI responses · GB10 / Blackwell · 2026-06-15 · source: tribev2-research ↗

TRIBE v2 maps video, audio, and text onto the cortical surface (20484 fsaverage5 vertices). We treat it as a subject in an fMRI experiment: feed it controlled stimuli, read out its predicted brain maps, and analyse them like a localizer study — no real brain data in the loop. New here? With no real brain data, the model recovers the localization of speech to primary + secondary auditory cortex. (The subtler sentences > word-lists syntactic contrast is weak here — near-identical TTS audio, low single-block power.)

② Does a cat-art finetune leave a "cat fingerprint"? no

An ASCII-cat LoRA (reproduced in bf16 from pookie3000/ascii-cats) swapped into the frozen text branch. Minimal-pair stimuli swap only the creature word.

  • Not cat-specific: a roughly uniform ~8% drift across all categories — dog moves more than cat.
  • Lands in language cortex: 62.9% of most-changed vertices are language network (5.4× enrichment), 0.0% visual. Both intuitive hypotheses — cat-selectivity, and a push toward visual cortex because ASCII art is "drawing" — were refuted.
  • Control passed: stock Llama through the exact finetune code path reproduces base to Δ = 0.004% (cosine 1.00000) — the 8% is genuinely the LoRA, not a cache/dtype artifact.

TRIBE's predicted maps are sensitive to the extractor's distribution but robust to its semantic specialization.  Full report →

③ Cross-modal: a hint that didn't survive scale

The video branch (V-JEPA 2) reads a real cat photo straight into visual cortex (5.48× enrichment), and a single iconic ASCII cat as text seemed to jump from 0.85× → 2.05× visual under the finetune — an exciting cross-modal bridge.

testfinding
img2ascii realism ladder (bitmap → V-JEPA)holds higher-res ASCII ≈ more cat-like (cos 0.55 full > reduced)
finetune convergence (ASCII-text → photo anchor)null gain −0.006 across 6 img2ascii cats (3/6 positive)

The photo-to-ASCII scale-up came back null — but it changed the stimulus distribution (line art → brightness-ramp mush), so it couldn't actually test the n=1 hint. Scaling the hint in its own regime (experiment ④) shows it was real after all — just generic.  Full report → · see the correction in ④ →

Reproducibility

Every number on the linked pages is recomputed from saved .npz maps, so the pages can't drift from the data. Each cross-modal effect is guarded by a null-swap control (stock model through the experimental code path must reproduce baseline to ~0). Built on PyTorch 2.12 + CUDA 13 for Blackwell sm_121; text branch meta-llama/Llama-3.2-3B, video V-JEPA 2, audio w2v-bert-2.0.
